# Dotfiles

A centralized location for all of my configs. Based off [Dotbot](https://github.com/anishathalye/dotbot#). 

## Installation
```bash
git clone https://github.com/rgrizzell/dotfiles.git ~/.dotfiles && ~/.dotfiles/install
```
